新内容

  • 版本记录: 2020年5月5日

  • 版本: 10.15.4 19E2269

Apple今天发布了macOS Catalina 10.15.4补充更新（19E2269）

新增对MacBookPro 2020的支持

今天令人惊讶的是，苹果发布了针对macOS Catalina 10.15.4的新补充更新。此更新的容量为1.01Gb，现在可用于所有10.15.4 Catalina用户。此更新旨在解决4个不同的问题。苹果还更新了BridgeOS，并重新发布了新的installer.app，组合更新和增量更新。

Catalina 10.15.4补充更新中包括什么？

macOS Catalina 10.15.4补充更新提高了Mac的稳定性和安全性。

  • 修复了运行macOS Catalina 10.15.4的Mac计算机无法与运行iOS 9.3.6和更低版本或OS X El Capitan 10.11.6和更低版本的设备参与FaceTime呼叫的问题
  • 解决了您可能反复收到Office 365帐户的密码提示的问题
  • 修复了MacBook Air（Retina，13英寸，2020）可能会在Setup Assistant中挂起，或者在断开并重新连接4K或5K外接显示器时出现的问题
  • 解决了Mac中的USB-C端口可能无法响应的问题

macOS Catalina正式版

Mac 上你喜爱的一切，都因 macOS Catalina 而变得更精彩。你可以享用两款全新媒体 app；用新的查找 app 定位遗失的 Mac；而你常用的 iPad app 现在也有了 Mac 版。你甚至可以将 iPad 当作第二个显示屏，扩展桌面空间。现在，无论做什么，你将拥有远超以往的体验

新内容

  • 版本记录: 2020年3月25日

  • 版本: 10.15.4 19E266

macOS Catalina 10.15.4引入了iCloud Drive文件夹共享，屏幕时间通信限制，Apple Music时间同步的歌词视图等。此更新还提高了Mac的稳定性，可靠性和安全性。

Finder

•从Finder共享iCloud Drive文件夹

•进行控制，以仅将访问权限限制为您明确邀请的人员，或者将访问权限授予具有文件夹链接的任何人

•有权选择谁可以进行更改和上传文件以及谁只能查看和下载文件

Screen Time

•沟通限制，用于控制您的孩子在一整天和停机期间可以与谁进行沟通和联系

•为孩子播放音乐视频的控制

Music

•Apple Music的时间同步歌词视图，包括通过单击歌词视图中的一行来跳到歌曲中您喜欢的部分的功能

Safari

•选择将Chrome密码导入到iCloud钥匙串中，以便在Safari和所有设备中轻松自动填充密码

•用于复制选项卡和关闭当前选项卡右侧的所有选项卡的控件

•兼容计算机上对Netflix内容的HDR播放支持

带有Apple Arcade的App Store

•通用购买支持可在iPhone，iPod touch，iPad，Mac和Apple TV上单个购买参与应用程序

•最近玩过的Arcade游戏出现在Arcade选项卡中，因此您可以继续在iPhone，iPod touch，iPad，Mac和Apple TV上玩

Pro Display XDR

•您可以通过从多个色域，白点，亮度和转印功能选项中进行选择，来定制适合特定工作流程需求的参考模式

辅助功能

•头部指针首选项，用于根据头部的精确移动在屏幕上移动光标

此更新还包括错误修复和其他改进。

•高动态范围输出到兼容HDR10的第三方显示器和与DisplayPort或HDMI连接的电视

•Outlook.com帐户的OAuth身份验证支持可提高安全性

•在辅助设备上升级到iCloud提醒时支持CalDav迁移

•解决了在启用“暗模式”时在应用之间复制的文本可能不可见的问题

•修复了Safari中验证码图块可能无法正确显示的问题

•解决了提醒可能会发送已完成提醒的通知的问题

•解决了从睡眠状态唤醒后LG UltraFine 5K显示屏的屏幕亮度问题

镜像靓点：

  • 抗击新型冠状病毒，人人居家，狂虐黑苹果

  • Clover默认配置文件config.plist原则上支持各种机型引导安装；

  • 新增CFL/KBL/SKL DVMT补丁

  • 更新CLOVER到v2.5k r5109【可能部分主题存在显示错位问题，介意者请使用r5107及以下版本】

  • 新增PE引导分区，同时支持CLOVER / PE双引导，三分区特别版同时支持CLOVER / OC / WEPE引导

  • 新增MBR引导，支持旧主板旧机型的安装

  • 驱动更新

    • 更新Lilu v1.4.3
    • 更新AppleALC v1.4.8
    • 更新WhatEverGreen v1.3.8，根本上解决黑屏问题
    • 更新USBInjectAll v0.7.4 增加新的USB主控

  • 去除USB端口限制，减少禁行发生的几率；增加Lilu崩溃的日志信息显示；

  • 本镜像采用官方原版app制作，集成Clover r5098，支持UEFI启动安装;apfs.efi同时去除日志显示；其它的驱动位于 drivers/off；

  • 300系列主板请于UEFI目录中移除AptioMemoryFix-64.efi添加OsxAptioFix2Drv-free2000.efi该驱动位于/EFI/CLOVER/drivers/off目录下 或者 Slide值获取及计算

  • Clover集成了本人修改的全新主题，不喜欢的可以于安装系统后自行替换；

  • 所有配置文件中默认添加Lilu崩溃日志的输出信息,详见：macOS Catalina 10.15安装中常见的问题及解决方法;

  • 更新WhateverGreen到v1.3.8(3月22日编译，根治安装/更新中的黑屏现象)，原生支持UHD620/UHD630等八代核显，不需要注入platform-id， 同时它也支持NVIDIA和AMD的显卡，以及整合了Shiki和CoreDisplayFixup的驱动，现在是All In One了；核显驱动更多的教程请参考：Hackintool(原Intel FB-Patcher)使用教程及插入姿势 / 黑苹果必备：Intel核显platform ID整理及smbios速查表 / Coffee Lake帧缓冲区补丁及UHD630 Coffee Lake ig-platform-id数据整理 教程：利用Hackintool打开第8代核显HDMI输出的正确姿势

  • 其它机型的配置文件统一保存到config-Other目录下，请自行复制相应机型配置文件；

    • 其它配置文件包括：HD520/530/540，HD550/P530,HD5000/5100/5200/5300/5500/6000,HD4200/4400/4600；所有config配置文件都可通过Clover引导界面-Options-configs进行选择；不会操作的请移步：Clover使用教程 macOS安装教程 Mojave硬件支持列表
    • HD3000驱动方法请移步这里
